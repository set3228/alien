Genre

Alien is a simple platformer where you can run, jump and kill everyone

Technologies

The game was built using JavaScript, Phaser, Webpack and Kostyli

Resolution

Resolution of the game screen is 1280 x 700, so if your device does not comply with this resolution, Alien will be sad

Goal

Collect stars, kill enemies and then reach finish
